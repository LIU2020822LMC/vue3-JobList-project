## 用法

---

这个项目使用JSON-Server作为模拟后端。

### 安装依赖

```bash
npm install
```

### 运行 JSON 服务器

服务器将在 [http://localhost:8000](http://localhost:5000/) 上运行

```bash
npm run server
```

### 运行 Vite 前端

Vue 将在 [http://localhost:5173](http://localhost:5173/) 上运行

```bash
npm run dev
```

### 生产构建

```bash
npm run build
```

### 预览生产构建

```bash
npm run preview
```
